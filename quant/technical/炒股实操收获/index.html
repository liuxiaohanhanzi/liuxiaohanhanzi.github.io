# 股票所属板块在哪看？板块情况怎么看？

在同花顺指数中，同花顺自己编制了各个板块，结构如下 ——

  • 同花顺指数
    • 同花顺概念
    • 同花顺行业
    • 同花顺地域（实用性弱一些）

板块主要用来看涨跌幅和资金净流入量

# 当日不及预期，次日处理方式

  1. 指数
  2. 板块
  3. 开盘三根交割单！！下杀第一时间跑，早跑！！！
  4. 绝对龙头，中期（4-7 板）可以等等
