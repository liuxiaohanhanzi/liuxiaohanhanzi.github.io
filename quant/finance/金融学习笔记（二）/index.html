# 爆仓

爆仓是在投资中，亏损到无法承受的程度，账户里的资金已经不足以继续持有你的投资了。特别是在用杠杆交易时，一旦账户里的钱不够覆盖亏损，你的仓位就会被强制平仓，这就是爆仓。

比如：你拿了一笔钱去投资，但是为了放大收益，你还借了一些钱。市场不如预期，亏损逐渐增加，最后你连本金都亏得差不多了，借的钱也快赔不起了，系统就会自动帮你卖掉所有的投资来止损，这就是爆仓。

# 杠杆

杠杆就是用少量的钱去撬动更大的投资，就像用杠杆撬动重物一样。它让你用自己的一部分资金，加上借来的资金，进行更大规模的交易。

杠杆工具主要有几种，常见的包括融资融券、期货、外汇保证金交易和差价合约（CFD）。

1. 融资融券

这是股票市场上的常用杠杆工具。融资就是你向券商借钱买股票，融券则是你借股票来卖空。如果股票涨了，融资能赚更多；融券则是希望股票跌。

2. 期货

期货是合约形式的交易，允许你用保证金（一部分钱）控制大额资产。

3. 外汇保证金交易

和期货类似，你用少量资金来控制更多的外汇投资，剩下的由平台借给你。汇率变化通常较小，杠杆让微小的价格波动也能带来较大的收益或损失。

# 期权

期权是一种合约，它给了你在未来某个时间以某个价格买入或者卖出某种资产的权利，但并不是义务；可以把它理解成买一个 “选择权”，而不是必须去做这笔交易。

期权分为两种：看涨期权和看跌期权。看涨期权是你有权在未来以一个确定的价格买入某个资产，如股票；看跌期权则是你有权以一个确定的价格卖出资产。

比如：你看好某个股票未来会上涨，现在价格是 100 块，你可以买一个看涨期权 —— 未来可以用 110 块的价格买入。如果股票真的涨到了 150 块，你用 110 块买入，然后在市场上卖 出，你就赚了。而如果股票没有涨，你就可以选择不买，最多损失的是你买期权的费用。

# 合约

合约（Future）其实就是一种协议，双方约定在未来某个时间，按照预定的条件买卖某种资产。常见的合约就是期货合约和期权合约。

合约在股票市场即期货（期权），叫法不同而已

期货合约是买卖双方都必须在未来某个时间，以约定的价格买入或卖出某种资产

期权合约给了你一个选择权，你有权决定是否执行这个合约，但不是必须的。

# 平仓

平仓就是结束你在某个交易中的持仓，也就是把之前的买入或卖出操作反向操作一次，从而关闭这个合约。

这个术语在合约交易中用得比较多

# 头寸

头寸（Position）是你在某个市场中持有的投资或交易量。它可以是买入的数量，也可以是卖出的数量。

多头头寸：如果你买了 100 股某公司股票，你就有了 100 股的多头头寸。你希望股票价格上涨，这样你可以通过卖出获利。

空头头寸：如果你卖空了 50 股股票，期望股票价格下跌，从而可以在未来以更低的价格买回，赚取差价。

与仓位不同的是，仓位往往指资产的配置百分比和风险暴露。

# 庄家策略

也就是砸盘，吸筹，拉升，出货

砸盘：指大规模卖出资产，导致价格迅速下跌。

吸筹：指在市场价格较低时，悄悄买入大量资产。

拉升：指通过大量买入资产，推动其价格上涨。目的是制造出上涨的假象以吸引散户投资者。

出货：指在价格上涨到一定水平后，大量卖出资产以实现利润。

# Lasso 和 Ridge 回归区别

Lasso：由于 L1 正则化的特性，Lasso 在处理多重共线性时会随机选择其中一个相关特征，并将其他相关特征系数设为 0。因此 Lasso 可以得到稀疏解；在特征高度相关时，其表现不稳定。

Ridge：L2 正则化不会使系数变为 0，它会缩小系数，但不会产生稀疏性。因此 Ridge 不会自动执行特征选择；在处理多重共线性时表现更好，可以保留所有相关信息。

# 期货

最早的期货市场诞生于农产品交易，比如农民和商人之间约定在未来某个时间以固定价格交付一定量的粮食。这种合约能避免因为市场波动导致的损失。这是期货的 “本意”，即作为一种风险管理工具来稳定生产交易。

然而随着金融市场的发展，期货不再仅仅是为了促进实体经济生产的工具，逐渐演变成了杠杆化的金融工具。很多交易者利用期货进行投机和短期交易，而不仅仅是为了对冲风险。
