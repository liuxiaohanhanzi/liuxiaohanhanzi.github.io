# 学生

# 未亡人

每次经历打击
我都会大病一场
卧病在床，意识朦胧
就像一部分的自己枯萎了

今天我才醒悟，
其实当时我就溺死在了过去
我的生活是漂浮的尸体投下的倒影
腐烂成夜晚那些五彩斑斓的梦

# 俗

来自知乎用户

“不食烟火的小孩子，看到中年妇女在菜市场大杀四方是会噘嘴的，而二十来岁的小姑娘，大多会偷偷跟着记笔记，看看大姨们怎么发挥。谁买到新鲜的桃子，谁就能高兴上多一个日子。

人从五谷不分的小毛孩儿成长到尝尽了柴米油盐的老掌柜，又有什么不好的呢？

诗读多了，日子过满，便不在乎那些虚头巴脑的了，让碎玉叮当响它的，也让烟火给榆木的桌子包它的浆。”

# 404

就像对老虎谈吃素
现在回忆起来
所谓温情，那一颦一笑
就像和人皮木偶逢场作戏
更多是画皮般的惊悚

# 褪色

来源淘宝：[之趣坊]

你问会褪色嘛？没办法，一定会的
一般金色是最容易掉色的，露出下面的银色喷砂；而机身会褪色成纯铜的玫瑰红色
所有的打火机使用后，都会磨损、掉漆、变色、图案氧化、贴章脱落

任何东西都会用旧，一件衣服会有洗掉色的时候，一个水杯会被磕碰变形
但是人不可能任何东西都不用，
日常生活就是这样啊

# 焦虑
