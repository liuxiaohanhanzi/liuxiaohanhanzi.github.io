# 文艺逼，请正常点

无论国内圈子还是洋人，如今文艺界创作者们不好好搞美学，一定要夹杂一堆自己的私货，也就是对社会的批判。

这样不是不好，因为任何艺术都有对现实的再加工，都一定会难以避免地带上批判的元素。问题是，这帮大脑空空的混子，没有一点沉实的思考，所讨论的议题全都是 “原生家庭”“家暴”“背叛出轨”“中式教育”“少数群体平权” 的流行单品，多少年了没一点新意……

在有些圈子里，家庭幸福好像搞得跟原罪似的

受伤的人需要一个墙角来相互安慰，这没问题。但有些人的音量，在互联网，真的太大了。抱团取暖的小众群体，给我有小众群体的自觉啊；那些意见领袖创作者给出的结论，要么就是逃避，要么就是纯粹的情绪宣泄。

好，不幸确实是创作者的缪斯，于是你们流行，你们解构的一地鸡毛，然后观众拍手叫好 “太美了”“太勇敢” 了，最后满地碎屑和狂欢的礼花，就像那些鲜艳但是干瘪的锡箔糖果纸。

你们是一群附着在主流共识的动脉上，吸血给自己当精神血包的瘾君子。我衷心地祝愿你们能像真正的嬉皮士一样，永远活不过三十岁。

# 应试语文

最近在看红楼梦。感慨啊，学生时代的我一没进入社会，二不会和人打交道，你让我读红楼梦我怎么可能读懂？

高中生的我对历史没时间了解，除了教科书一窍不通，你让我读文言文，和我讲江陵公、讲宋濂，我又怎么可能有深刻理解？

为了应试作文，初中时整天写花草树木，所谓托物言志实为无病呻吟天天开悟；高中时只知道唱高调立牌坊，造一堆无根楼阁浮于空中不接地气

# 癸酉本

字字看来皆是血，好好一本红楼梦被搬套为曹家家事，被扭曲为小家子气的抄家故事。

真本，是有服人的魅力的。红楼梦绝对是悼明之亡，揭清之失。按这个思路一切都通畅了，红楼梦里的森森鬼气也可解释，甚至令人垂泪了。

在我这里，人生三大遗憾，只剩其二。
