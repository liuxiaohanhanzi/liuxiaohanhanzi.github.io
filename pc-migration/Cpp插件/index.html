# EasyX

感谢 Limia40 的教程

博主直接使用 CLion 自带的 MinGW，操作如下

  1. 打开官网，点击右上角更新日志，下载 EasyX_for_MinGW
  2. 解压
  3. 将 include 文件夹下的 easyx.h 和 graphics.h 拷贝到 CLion\bin\mingw\x86_64-w64-mingw32\include
  4. 将 lib64 下的 libeasyx.a 拷贝到 Clion\bin\mingw\x86_64-w64-mingw32\lib
